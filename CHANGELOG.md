# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2026-05-29

### Added
- Support for @quasar/app-vite v2.x
- Maintain backward compatibility with @quasar/app-vite v1.x

### Changed
- Updated compatibility check to accept both @quasar/app-vite v1 and v2

### Compatibility
- quasar: ^2.0.0
- @quasar/app-vite: ^1.0.0 || ^2.0.0
- @quasar/app-webpack: ^3.0.0

### Migration Notes
No breaking changes. Users on @quasar/app-vite v1 can continue to use the extension.
Users upgrading to @quasar/app-vite v2 should update to this version.
